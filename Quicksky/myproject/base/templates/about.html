About QuickSky Flights ✈️

QuickSky is a modern flight booking platform that allows users to search, book and manage airline tickets easily.

This project is built using Django, HTML, CSS and provides features like booking, history, profile management and secure login.

Our goal is to provide a smooth and fast flight booking experience, just like real airline websites.
